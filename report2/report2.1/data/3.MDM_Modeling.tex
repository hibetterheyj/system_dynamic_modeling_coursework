\begin{equation}\label{p2}

\end{equation}
$e_1$是输入变量，一直保留在最终的方程中；$e_3$来自阻性元件$R _ { eL }$的输出；$e_4$直接与状态变量有关，故而有下面三式：

\begin{equation}\label{e3}

\end{equation}

由式(\ref{p2})-(\ref{e4})得到状态方程1：
\begin{equation}
\dot { p } _ { 2 } = V _ {L}  - \frac { p _ { 2 } } { I _ { eL}  } R _ { eL} - k _ { 1 } \frac { p _ { 6 } } { J _ { rL} },
